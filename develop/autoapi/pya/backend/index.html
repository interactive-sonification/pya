Determine a suitable Backend implementation

This will first try a local pyaudio Backend unless force_webaudio is set.

Parameters:

  • force_webaudio (bool, optional) – prefer JupyterBackend, by default False

  • port (int, optional) – port for the JupyterBackend, by default 8765

Returns:

Backend instance

Return type:

PyAudioBackend or JupyterBackend

Raises:

RuntimeError – if no Backend is available
